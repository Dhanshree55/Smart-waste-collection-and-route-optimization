♻️ Smart Waste Management Dashboard

An AI-powered waste monitoring and optimization system built using Streamlit, Machine Learning, and Data Visualization. This project simulates smart city waste management with route optimization, forecasting, and performance analytics.

🚀 Features
📊 Interactive Dashboard (Streamlit UI)
🤖 Machine Learning Forecasting (Random Forest)
🚛 Waste Collection Route Visualization
📈 KPI Monitoring
Total Waste
Average Waste
Route Efficiency
Forecast Waste

🌱 Sustainability Metrics
Carbon Footprint
Segregation Rate
Energy Generation
🛠 Issue Tracking System
💰 Profit Analysis Visualization

🧠 Tech Stack
Frontend/UI: Streamlit
Backend Logic: Python
Libraries:
Pandas
NumPy
Matplotlib
Scikit-learn
ML Model: Random Forest Regressor
